5. Write a program to find the circumference of a circle with radius 10 in C++.
Ans:
#include<iostream>
using namespace std;
int main()
{
    int r = 10;
    float cir = 2*3.1415*r;
    cout<<"circumference of the circle is "<<cir;
}
